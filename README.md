# NanoSpeed ⚡

A high-performance LLM inference engine built from scratch in JAX, optimized for both research and education.

## 🎯 Goals

- **Educational**: Understand how LLM inference works at a deep level
- **Fast**: Competitive performance with production systems
- **JAX-native**: Leverage JAX's functional paradigm and JIT compilation
- **Research-ready**: Clean, hackable code for experimentation
